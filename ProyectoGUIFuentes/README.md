# FADA_II_SUBJECT_PROJECT_II

#### Descripción

Esta interfaz gráfica permite:
- Cargar parámetros de una instancia del problema MinPol
- Generar automáticamente el archivo instancia_generada.dzn
- Descargar el .dzn
- (Opcional) Enviarlo a un backend para ejecutar MiniZinc
-Visualizar resultados: polarización, mediana y matrices

Desarrollada en:

- React 18
- Vite
- TailwindCSS
- Framer Motion
- Lucide Icons

# 1. Requisitos
Asegurarse de tener instalado:

| Software | Versión recomendada  |
| -------- | -------------------- |
| Node.js  | ≥ 18.0               |
| npm      | ≥ 9.0                |
| MiniZinc | ≥ 2.7 (solo backend) |

# 2. Instalación
Clonar repositorio

´´´
git clone https://github.com/JhonierM14/FADA_II_SUBJECT_PROJECT_II
cd ProyectoGUIFuentes
´´´

Instalar dependencias:
´´´
npm install
´´´

# 3. Ejecutar en modo desarrollo

´´´
npm run dev
´´´

Abrir en el navegador

´´´
http://localhost:5173
´´´
